# Project 03 — Final Project Proposal: Arnold Archaga Portfolio

This repository contains a complete project proposal + a two-page website prototype for a personal portfolio website. It includes project management documentation (scope, plan, retrospective) and a functional prototype built with semantic HTML and a shared external stylesheet.

## Live Website
- Live site: https://arnoldarchaga.github.io/project-03-final/

## Documentation
All planning documents are located in the `docs/` folder:
- `docs/scope.md` — Project Scope Statement (revised)
- `docs/plan.md` — Project plan (TAME risk analysis + Work Breakdown Structure)
- `docs/retrospective.md` — Project retrospective

## Website Pages
- `index.html` — Home / overview
- `about.html` — About / background & goals
- `style.css` — Shared stylesheet for both pages

## Tech
- HTML5 (semantic structure)
- CSS (single external stylesheet)
- GitHub Pages (deployment)

## How to Run Locally
1. Download or clone the repository
2. Open `index.html` in your browser
